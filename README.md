# BetterLib - 对浙江大学图书馆系统的增强实现

## 介绍

BetterLib 是一个基于浙江大学图书馆系统的增强实现，旨在提供更丰富的功能和更友好的用户体验。

## 已实现的功能/优化

- 通过对用户登录信息的安全存储，实现了自动登录功能，用户完成一次登录后，后续访问无需再次输入用户名和密码。
- 对空间预约条目做了信息优化，增加了更详细的区域特点描述，如方位和具体配置信息。
- 支持在首页查看预约信息时显示预约位置的方位（目前较简略）。
- 实现座位的列表预约，可视化待实现。
- 重构了预约座位的地图，在地图上添加了如书架和窗户等更详细的信息。
- 在预约座位的条目预览中直接显示座位的方位信息，比原本的标示图更清晰。
